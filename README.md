# 100Hires AI Tools Setup

## Overview

This repository documents the first step of the 100Hires portfolio project process.

The goal of this step was to install the required tools, create a public GitHub repository, open it in Cursor, document the setup process, and push the README file to GitHub.

## Tools Installed

* Cursor IDE
* Claude Code extension/add-on in Cursor
* Codex extension/add-on in Cursor
* Git
* GitHub

## Steps Completed

1. Installed Cursor IDE.
2. Opened Cursor and accessed the editor window.
3. Opened the Extensions section in Cursor.
4. Searched for and installed the Claude Code add-on.
5. Searched for and installed the Codex add-on.
6. Created a public GitHub repository.
7. Cloned the repository locally.
8. Opened the repository in Cursor.
9. Created this README.md file.
10. Documented the setup process.
11. Committed the README.md file using Git.
12. Pushed the repository to GitHub.

## Issues Encountered

At first, I was in the Cursor Agents interface and could not find the Extensions section.

## How I Solved Them

I solved this by opening the full Cursor editor window. After that, I was able to access the Extensions section, search for the required add-ons, and install them.
I also used Git from the terminal to commit and push the README file to the public GitHub repository.

## Result

The setup process was completed successfully.

The repository is public and this README documents the tools installed, the steps completed, the issues encountered, and how they were solved.
